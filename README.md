# AirAware – Smart Air Quality Prediction System

## Problem Statement
Air pollution has become a serious concern affecting both human health and the environment. Predicting air quality in advance can help individuals and authorities take timely preventive measures.
## Objective
To predict air quality levels using machine learning techniques based on historical air pollution data.

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Dataset
The dataset consists of historical air quality data containing the following parameters: PM2.5, PM10, NO2, SO2, CO.

## System Architecture
Input Data → Preprocessing → ML Model → AQI Prediction → Output

## How to Run
pip install -r requirements.txt  
python app.py

## Future Scope
- Real-time AQI prediction
- API integration
- Mobile application

## Conclusion
AirAware provides a simple and effective approach for predicting air quality using machine learning. This system can help improve awareness and support decision-making for a healthier environment.
